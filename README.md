# ИИ-ассистент для компании по организации свадеб

## Описание проекта

Проект представляет собой интеллектуального ассистента, разработанного для автоматизации взаимодействия с клиентами компании, занимающейся организацией свадеб. Ассистент способен вести осмысленный диалог, отвечать на вопросы клиентов, основываясь на базе знаний компании, а также записывать клиентов на встречи и добавлять их в Google Календарь.

Цель проекта — упростить процесс коммуникации между компанией и клиентами, сократить время на рутинные задачи и повысить качество обслуживания за счет быстрых и точных ответов.

---

## Функциональность

### 1. Осмысленное ведение диалога
Ассистент умеет поддерживать естественный диалог с клиентами, используя современные технологии обработки естественного языка (NLP). Он способен понимать контекст вопросов и давать релевантные ответы.

### 2. Ответы на вопросы по базе знаний
Ассистент имеет доступ к базе знаний компании, которая содержит информацию о предоставляемых услугах, пакетах услуг, ценах, условиях сотрудничества и других важных аспектах. Это позволяет ему предоставлять точные и актуальные ответы на вопросы клиентов.

### 3. Запись на встречу
Клиент может запросить запись на встречу через ассистента. Для этого достаточно указать удобное время и дату. Ассистент проверяет доступность слотов и подтверждает запись.

### 4. Интеграция с Google Календарем
Все записи на встречи автоматически добавляются в Google Календарь компании. Это обеспечивает синхронизацию расписания и минимизирует риск двойного бронирования.

---

## Используемые сервисы

### 1. **Qwen**
- **Назначение**: Создание системного промпта и базы знаний.
- **Описание**: Qwen используется для формирования структурированной базы знаний, которая служит основой для ответов ассистента. Также с его помощью разработаны системные промпты, определяющие поведение ассистента.

### 2. **Савви (Suvvy.ai)**
- **Назначение**: Создание ИИ-ассистента.
- **Описание**: Платформа Suvvy.ai предоставляет инструменты для создания и настройки ИИ-ассистента. Она обеспечивает интеграцию с внешними сервисами, такими как Google Календарь, и поддерживает функционал для ведения диалога.

---

## Преимущества решения

- **Экономия времени**: Автоматизация рутинных задач, таких как ответы на часто задаваемые вопросы и запись на встречи.
- **Повышение качества обслуживания**: Быстрые и точные ответы на вопросы клиентов.
- **Удобство для клиентов**: Возможность получить информацию и записаться на встречу в любое время суток.
- **Синхронизация данных**: Интеграция с Google Календарем исключает ошибки при планировании встреч.
